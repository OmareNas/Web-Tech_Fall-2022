Web Technologies: List, Links and Images

Lists

    Ordered Lists:
  1. List item position important
  2. Numbered or alphabetized
  3. Steps to follow
  4. <ol> -start tag
  5. <ol/> -end tag
    6. List item:
    • All browsers indent list items by default
    • <li> -start tag
    • <li/> -end tag

    Unordered Lists:
  • List item position unimportant
  • Bullet points
  • <ul> -start tag
  • <ul/> -end tag
    • All browsers indent list items by default
    • <li> -start tag
    • <li/> -end tag


    Nested Lists

  • Used when a list within a list is required
  • By default, browser will
    • Indent the second list
    • Use a different bullet style
